# Identifiers in Python

# Definition

Identifiers are the **names used to identify variables, functions, classes, objects, or modules** in Python.

In simple words:

> Identifiers are names given to programming elements.

---

# Examples of Identifiers

```python id="w5e5sp"
name = "Rahul"
age = 21

def display():
    print("Hello")
```

Here:

* `name` → identifier
* `age` → identifier
* `display` → identifier

---

# Rules for Naming Identifiers

## 1. Must Start with a Letter or Underscore

✅ Correct

```python id="g0u8rn"
name = "Python"
_age = 20
```

❌ Wrong

```python id="8yzwmg"
1name = "Python"
```

Python does not allow identifiers to start with numbers.

---

# 2. Cannot Contain Special Symbols

❌ Invalid

```python id="4tb9s3"
student@name = "Ram"
```

Only:

* Letters (`A-Z`, `a-z`)
* Numbers (`0-9`)
* Underscore (`_`)

are allowed.

---

# 3. Spaces Are Not Allowed

❌ Wrong

```python id="i2g1bg"
student name = "Raj"
```

✅ Correct

```python id="g2m4ol"
student_name = "Raj"
```

---

# 4. Python Keywords Cannot Be Used

Keywords already have predefined meanings.

❌ Wrong

```python id="ovjlwm"
class = 10
```

Because `class` is a keyword.

---

# 5. Identifiers Are Case Sensitive

Python treats uppercase and lowercase differently.

```python id="x7u5ju"
name = "Ram"
Name = "Shyam"

print(name)
print(Name)
```

Both are different identifiers.

---

# Naming Conventions in Python

## Variables and Functions

Use lowercase with underscore.

```python id="49ijoe"
student_name
calculate_total()
```

---

## Classes

Use PascalCase.

```python id="jlwm64"
class StudentDetails:
    pass
```

---

# Real-Time Example

```python id="8iv6dh"

print(student_age)
```

Here:

* `student_name` → identifier
* `student_age` → identifier

These identifiers help store and access data.

# Keywords vs Identifiers

| Keywords                | Identifiers             |
| ----------------------- | ----------------------- |
| Reserved words          | User-defined names      |
| Cannot be used as names | Created by programmer   |
| Example: `if`, `while`  | Example: `age`, `marks` |

---

Output:


Because `if` is a keyword.
